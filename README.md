sw-festival-core
====

```
메시지 도착하면:
  만약 메시지 = 안녕:
    메시지 보내기 (안녕)
```

sw-festival-core는 2019 광주 SW 체험 축전 참여를 위해 개발된 코드 컨버터입니다.

지정된 한글 문자열을 파이썬 코드로 변환하여 프로그래밍을 좀 더 쉽게 할 수 있게 돕습니다.

>-숭덕고등학교 IT동아리, 404 컴퓨터 못 찾음에 의해 개발됨

## 시작하기

### 환경구성
Windows
```
pip install discord.py
python app.py
```

Linux
```
pip3 install discord.py
python app.py
```
